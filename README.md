website front-end for environmental impact score generation

    ├── README.md
   	├── webpack.config.Js

### Installation:
- Install packages: `npm i` or `yarn`
- Start development server: `npm run start` or `yarn start`
>Runs at port `8080`
- Generate deploy bundle: `npm run build` or `yarn build`
